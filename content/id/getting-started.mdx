# Memulai

Semua yang Anda butuhkan untuk menyiapkan template ini untuk proyek Anda sendiri.

## Prasyarat

- **Node.js** v18 atau lebih tinggi
- **pnpm** (direkomendasikan) atau npm/yarn

## Instalasi

<Steps>
### Salin template ini

```bash
cp -r nextra4-starter nama-docs-saya
cd nama-docs-saya
```

### Install dependensi

```bash
pnpm install
```

### Sesuaikan metadata situs

Edit `app/layout.tsx` dan perbarui:

```tsx
export const metadata: Metadata = {
  title: {
    default: 'Nama Situs Anda',
    template: '%s – Nama Situs Anda',
  },
  description: 'Deskripsi situs Anda',
  metadataBase: new URL('https://situsanda.com'),
}
```

### Perbarui sidebar

Edit `content/id/_meta.js` (Indonesia) dan `content/en/_meta.js` (English):

```js
export default {
  "index": "Beranda",
  "halaman-saya": "Halaman Saya",
  "bagian-saya": "Bagian Saya",
}
```

### Mulai menulis konten

Buat file `.mdx` di dalam `content/id/` (bahasa Indonesia) dan `content/en/` (English):

| File | URL |
| :--- | :--- |
| `content/id/index.mdx` | `/` atau `/id` |
| `content/en/index.mdx` | `/en` |
| `content/id/tentang.mdx` | `/tentang` |
| `content/en/about.mdx` | `/en/about` |

### Jalankan server pengembangan

```bash
pnpm dev
```

Situs Anda sekarang aktif di `http://localhost:3000`!
</Steps>

## Build untuk Produksi

```bash
pnpm build:search
pnpm start
```

<Callout type="info">
Perintah `pnpm build:search` menjalankan `next build` diikuti `npx pagefind` untuk membuat indeks pencarian. Jalankan setiap kali menambahkan konten baru.
</Callout>

## Tentang Dukungan Dua Bahasa

Pengguna dapat berpindah bahasa menggunakan dropdown di navbar. Konten setiap bahasa dipisah di folder masing-masing:

```
content/
├── id/    ← Konten bahasa Indonesia
│   ├── _meta.js
│   └── *.mdx
└── en/    ← Konten bahasa Inggris
    ├── _meta.js
    └── *.mdx
```

<Callout type="warning">
Setiap halaman perlu ada di kedua folder (`id/` dan `en/`) agar dapat diakses di kedua bahasa.
</Callout>
