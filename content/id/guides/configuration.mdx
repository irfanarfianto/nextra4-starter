# Konfigurasi

Contoh sub-halaman lainnya. Ganti konten ini dengan milik Anda sendiri.

## Contoh Tabel Konfigurasi

| Opsi | Tipe | Deskripsi |
| :--- | :--- | :--- |
| `opsi_satu` | `string` | Deskripsi opsi pertama |
| `opsi_dua` | `boolean` | Deskripsi opsi kedua |
| `opsi_tiga` | `number` | Deskripsi opsi ketiga |

## Contoh dengan Tabs

<Tabs items={['Opsi A', 'Opsi B']}>
  <Tabs.Tab>
    Konten untuk Opsi A. Gunakan tabs saat menampilkan alternatif.
  </Tabs.Tab>
  <Tabs.Tab>
    Konten untuk Opsi B.
  </Tabs.Tab>
</Tabs>
