# Welcome to My Docs 👋

> **Replace this with your own introduction.** This is the homepage of your documentation site.

This template is built with **Nextra 4** and **Next.js App Router**. It includes everything you need to start writing documentation immediately — including bilingual support (🇮🇩 Indonesian & 🇬🇧 English).

## What's Included

- ✅ Nextra 4 + Next.js App Router setup
- ✅ TypeScript support
- ✅ Bilingual support (Indonesian & English)
- ✅ Pagefind full-text search (after `pnpm build:search`)
- ✅ Sidebar navigation with `_meta.js`
- ✅ All Nextra components ready to use

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build with search index
pnpm build:search
```
